  - 다이소 특집, 지금까지의 나레이션과 마찬가지로 나레이션 오디션 받음

  - 일단은 듣고 이해하기 좋은 목소리로 말하는 것을 기본으로, 괜히 과장하거나 목소리를 만들지 않음
